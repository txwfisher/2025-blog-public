在 GitHub 上找项目、下代码，有时候会被英文界面卡住，或者下载速度慢得让人抓狂。其实只要装两个油猴脚本，就能让这些麻烦事少很多。

一个是 GitHub 中文化插件，把大部分界面直接变成中文，看项目、读文档都轻松不少；另一个是 GitHub 增强-高速下载，给克隆和 Release 下载加上加速通道，国内网络也能快不少。

下面简单说说它们的用法和特点，按需选择就行。

🇨🇳 GitHub 中文化插件

将 GitHub 的菜单、按钮、提示等界面元素汉化，并将项目描述一键翻译，阅读体验会友好很多。

安装步骤

1. 安装脚本管理器 "Tampermonkey" (https://www.tampermonkey.net/)。
2. 安装脚本："GitHub 中文化插件" (https://greasyfork.org/zh-CN/scripts/435208)
3. 刷新 GitHub 页面即可看到汉化效果。

🚀 GitHub 增强 - 高速下载

为 Git Clone / SSH、Release、ZIP 包、Raw 文件等添加多个高速下载入口，在国内网络下效果明显。

安装步骤

1. 确保已安装 Tampermonkey。
2. 安装脚本："GitHub 增强 - 高速下载" (https://greasyfork.org/zh-CN/scripts/412245)
3. 进入任意仓库，即可在下载区域看到新增的“高速下载”按钮。

用上这两个脚本，GitHub 的浏览和下载都会顺手很多。如果你平时经常在上面找项目，不妨试试。
