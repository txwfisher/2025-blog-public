## 🎯 插件一：GitHub 中文化插件 | 让代码阅读零压力

**🌟 核心亮点**  
瞬间将 GitHub 的英文界面转换为亲切的中文，让你专注于代码本身，而不是在语言理解上耗费精力。

**✨ 你会爱上这些改变**  
✅ **界面全面汉化**：所有菜单、按钮、提示都变成熟悉的中文  
✅ **智能翻译**：项目描述、README 文档支持一键翻译  
✅ **无感切换**：就像在使用国内的代码平台一样流畅自然  
✅ **保持原味**：代码部分不受影响，专业术语精准保留

**🚀 极简安装指南**  
1. **安装脚本管理器**（一次安装，终身受益）  
   在浏览器中搜索并安装 **Tampermonkey**，或直接访问：
   📦 [Tampermonkey 官网](https://www.tampermonkey.net/)

2. **获取汉化脚本**  
   打开脚本页面，点击醒目的绿色安装按钮：
   🌐 [GitHub 中文化脚本](https://greasyfork.org/zh-CN/scripts/435208)

3. **立即生效**  
   安装完成后，刷新任意 GitHub 页面，瞬间开启中文新世界！
